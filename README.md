# 启动

### iOS on Mac

- 参照[React Native文档]安装相关工具
  - Xcode (包含命令行工具)
  - Watchman
  - Node
  - `react-native-cli`
- 安装依赖
  - 首先安装 yarn 工具: `npm install yarn -g`
  - 通过 yarn 安装依赖: `yarn install`
- 执行 `react-native run-ios`
